# Public Assets

Static asset host for Tian's projects — used as a zero-cost public CDN via `raw.githubusercontent.com`.

## Structure

| Directory | Purpose |
|---|---|
| `sweetbowl/` | Sweetbowl Tokyo — product photos, website assets, brand kit |
| `chopchop/` | ChopChop — public-facing proposals and brand materials |
| `whill/` | WHILL APAC — promotional content |
| `tian/` | Personal profile — avatars, signatures |
| `notion-covers/` | Notion page cover images (external URL host) |
| `claude-skills/` | Demo screenshots for shared tooling |

## Usage

Embed any file directly:

```
https://raw.githubusercontent.com/Aetostian/tianyiyan-public-assets/main/<path>
```

Example:
```
https://raw.githubusercontent.com/Aetostian/tianyiyan-public-assets/main/sweetbowl/cookbook/sb-cookbook-01.jpg
```
